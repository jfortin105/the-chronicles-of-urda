# Learning
- You can use [[Mechanics and Rules/Downtime\|Downtime]] to learn a new skill
- Learning enables you to do new actions or gives you advantage on certain checks
- In order to lean a new skill, you must find a capable instructor who is willing to instruct you
- You can't typically learn another class's or ancestry's unique talents, but you can learn auxiliary skills
	- Examples of auxiliary skills
		- new language
		- use of specific weapon
		- travel using boat
		- scavenging for rations
		- etc


# Learning Check
- DC 18 INT
- Success:
	- Learn the new skill
- Failure
	- You can try again as your next downtime activity
	- When you next attempt learning this skill, lower the DC of the INT check by 1 (cumulative)
